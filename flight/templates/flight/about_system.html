О системе

Введение

Сервис бронирования авиабилетов «На взлёт!» создан для того, чтобы сделать процесс поиска и бронирования билетов на авиарейсы максимально удобным и приятным.

Сервис использует систему управления баз данных SQLite.

Системные требования: операционная система -- Windows 7 и новее; браузер -- 64-битный на базе Chromium (например, Google Chrome или Яндекс.Браузер).

Работа с системой

  • Главная страница
    На главной странице пользователь может указать пункт отправления искомых рейсов, пункт назначения, желаемый класс обслуживания и дату полёта. Поиск осуществляется нажатием кнопки «Найти рейсы». Если введённые данные некорректны, пользователю выведется сообщение об ошибке, и предоставлена возможность изменить введённые данные, в ином случае пользователь переходит на страницу просмотра списка найденных рейсов. В «шапке» (верхней части) каждой страницы расположено навигационное меню.
  • Авторизация
    Для входа в систему необходимо перейти на страницу авторизации по кнопке «Авторизация» в «шапке». На этой странице пользователь может пройти авторизацию, введя свои учётные данные -- логин и пароль, и нажав на кнопку «Войти». В случае неверно введённых данных будет выведено сообщение об ошибке и предоставлена возможность изменить введённые данные. Если пользователь ещё не зарегистрирован на странице, предусмотрена гиперссылка «Зарегистрируйтесь», открывающая страницу регистрации. Нажатие на логотип «На взлёт» возвращает на главную страницу.
  • Регистрация
    На странице регистрации пользователь может зарегистрироваться в системе, введя свои данные и нажав на кнопку «Зарегистрироваться». При правильно введённых данных откроется главная страница, а пользователь будет авторизован. В ином случае будет выведено сообщение об ошибке с указанием правильного формата данных, после чего пользователь сможет исправить введённые данные.
  • Просмотр списка рейсов
    Пользователь может просматривать список доступных рейсов, соответствующих его запросу, на странице результатов поиска. Здесь пользователь может фильтровать найденные рейсы по цене, времени вылета и времени посадки, сортировать по убыванию и возрастанию цены места. Если пользователь авторизован, он может перейти к странице бронирования, нажав на кнопку «Забронировать», в ином случае откроется страница авторизации.
  • Бронирование
    На странице бронирования представляется детальная информация о рейсе. Пользователь может забронировать место на рейс, указав контактную информацию (телефон и адрес электронной почты), фамилию, имя, отчество и пол пассажира, а также выбрав место на схеме салона самолета. Красным цветом отображаются забронированные другими пользователями места, серым -- места, выбранные в рамках текущего бронирования, зелёным -- свободные (доступные к выбору) места. Для бронирования пользователь может добавить несколько пассажиров. Оформить бронь при правильно введённых данных можно по нажатию кнопки «Перейти к бронированию». Если данные введены неверно, выводится сообщение об ошибке. После оформления брони пользователь увидит страницу подтверждения бронирования. На ней пользователь может загрузить документ о бронировании в формате PDF, нажав на кнопку «Скачать бронь».
  • История бронирований
    Все бронирования пользователя можно посмотреть на странице истории бронирований, где показывается информация о рейсах, а также статус бронирования. Пользователь может отменить бронь нажатием кнопки «Отменить», если до вылета рейса осталось более 6 часов, либо загрузить документ о бронировании в формате PDF, нажав кнопку «Скачать».
  • Сведения о системе
    Пользователь также может открыть страницу «О системе», что Вы уже и совершили. Здесь представлена информация о функциональных возможностях сайта и описаны все ключевые функции системы.
  • Сведения о разработчиках
    Посмотреть информацию о команде разработчиков, реализовавших сервис, можно, нажав кнопку «О разработчиках» в «шапке» страницы.
  • Выход из аккаунта
    Выход пользователя из аккаунта осуществляется по нажатию кнопки «Выйти» в выпадающем меню «Мой аккаунт», которое расположено в правой части «шапки» страницы. Также можно перейти на страницу личного кабинета, нажав на кнопку «Личный кабинет».
  • Личный кабинет
    На странице личного кабинета можно просматривать и редактировать личные данные: имя, фамилию, отчество, номер телефона, электронную почту и пароль. Чтобы появилась возможность изменять данные, необходимо нажать на кнопку «Изменить данные». Если данные введены корректно, по нажатию на кнопку «Сохранить» будет осуществлено сохранение данных и обновление страницы. Иначе будет показано сообщение о некорректности введенных данных.
